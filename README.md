# Chinni's Memory Match Puzzle Game

## 🧩 Overview

Welcome to **Chinni's Memory Match**, an interactive browser-based puzzle game designed for fun and brain-teasing! This game is perfect for kids and adults alike, challenging your memory and concentration as you uncover matching pairs of emojis.

## ✨ Features

*   **Interactive UI:** A standalone web application with a user-friendly graphical interface (not text-based).
*   **Multiple Levels:** Progress through 5 exciting levels, each increasing in difficulty with more pairs to match.
*   **Star Rating System:** Earn 1 to 3 stars at the end of each level based on your score, adding a fun competitive element.
*   **On-Page Celebrations:** No more intrusive alerts! Enjoy subtle on-page visual celebrations (like fireworks emojis) and celebratory sounds when you complete a level.
*   **Sound Toggle:** Easily turn game sounds on or off.
*   **Restart Functionality:** Restart the current level anytime.
*   **Mobile-Friendly:** Designed with responsive CSS to play smoothly on various screen sizes, from mobile phones to desktops.
*   **Simple & Engaging:** Easy to understand rules with delightful emoji cards.

## 🚀 How to Play

1.  Click the "Start Game" button on the main screen.
2.  The game board will display face-down cards.
3.  Click on two cards to flip them over.
4.  If the cards have matching emojis, they will stay face-up and turn green.
5.  If they don't match, they will flip back down after a short moment.
6.  Match all the pairs to complete the level!
7.  Your score and earned stars will be displayed, and you can proceed to the next level or restart.
